# Normalizer

Um simples normalizador de texto utilizado para gerar
nomes para arquivos de classe seguindo as
convenções de nome do C#. Feito sem a utilização de Regex,
utilizando as classes Normalize e CharUnicodeInfo para
tratar o texto.

A ideia é utilizar o enunciado do exercício
como nome do arquivo .cs para ficar fácil de encontrar depois.

E.g. 

Colando o texto:
Otimização de Alocação de Ativos com Restrições

O console devolve pra tela e pra área de transferência:
OtimizacaoDeAlocacaoDeAtivosComRestricoes
